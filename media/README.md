## Adding or replacing media

- Prefer a descriptive filename for new assets.
- Add useful alt text at every Markdown reference.
- Keep screenshots free of credentials, private hostnames, and customer data.
- Compress large PNGs when readability is not affected.
- Do not rename an existing hashed file without updating all references.
- Keep editable source projects separate from final presentation/video assets.
