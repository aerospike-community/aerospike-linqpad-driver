# Bundled Runtime Dependencies

This is a self-contained LINQPad driver package. The following runtime
components are bundled under `lib/net8.0` and are not restored as separate
NuGet dependencies.

| Package | Version |
|---|---:|
| Aerospike.Client | 8.4.2 |
| GeoJSON.Net | 1.4.1 |
| NeoLua | 1.3.19 |
| NetEscapades.Configuration.Yaml | resolved version |
| Newtonsoft.Json | 13.0.4 |
| YamlDotNet | resolved version |

LINQPad.Reference is used only at compile time and is supplied by LINQPad.
